# Eyeson
Internal eyeson api rails sdk for service app implementation

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'eyeson'
```

And then execute:
```bash
$ bundle
```

## Configuration
```ruby
Eyeson.configure do |config|
  config.endpoint = 'https://api.eyeson.team'
  config.
end
```
